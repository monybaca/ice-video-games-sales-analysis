# ice-video-games-sales-analysis
Análisis integral de ventas de videojuegos para la tienda online ICE, evaluando plataformas, géneros, puntuaciones de críticos y usuarios, así como tendencias en regiones clave. Incluye limpieza de datos, análisis exploratorio, selección del período relevante y comparación de mercados para identificar plataformas prometedoras y géneros rentables
